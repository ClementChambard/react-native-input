PasswordStrengthMeter Component Documentation

Usage


import { PasswordStrengthMeter } from 'react-native-input';

// Example usage
<PasswordStrengthMeter
  placeholder="Enter your password"
  onChangeText={(password) => console.log('Password changed:', password)}
/>

Customization

Styling

The component provides various styling options for customization.

  • textInputStyle: Style for the text input.
  • progressBarStyle: Style for the password strength progress bar.
  • containerStyle: Style for the container that wraps the component.
  • messageStyle: Style for the password strength message text.

Password Strength Calculation

By default, the component uses a simple strength calculation function. You can customize the strength calculation by providing your own function through the strengthFunction prop.

Visual Feedback

The component provides visual feedback on the password strength through a progress bar and a strength message. You can customize the color gradient and messages using the colorGradient and messages props.

Example


<PasswordStrengthMeter
  placeholder="Enter your password"
  onChangeText={(password) => console.log('Password changed:', password)}
  textInputStyle={{ borderColor: 'blue' }}
  progressBarStyle={{ backgroundColor: 'green' }}
  containerStyle={{ margin: 10 }}
  messageStyle={{ fontStyle: 'italic' }}
/>

Feel free to further customize the component based on your application's requirements.
